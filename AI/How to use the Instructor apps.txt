To use the instructor apps provided during the course ...

1: Unzip the Instructor App to your Qlik Sense Apps folder

2: Rename the file extension from .qvfbak to .qvf 

3: If using a browser to access the Hub then refresh the browser and
   the app will show up

4: If using the desktop (i.e Not using a browser) then shutdown Qlik Sense
   and restart it as sometimes the HUB does not refresh properly.
